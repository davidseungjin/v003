---
description: Default technical stack and file conventions for web design recreation tasks
---

- Use Tailwind CSS via CDN: `<script src="https://cdn.tailwindcss.com"></script>`
- Use `https://placehold.co/` for placeholder images when source images aren't provided
- Mobile-first responsive design
- Single `index.html` file with all content inline unless the user requests otherwise
- Serve locally with `npx serve` or equivalent for Playwright screenshots (file:// URLs are blocked)
