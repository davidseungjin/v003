# Website Design Recreation

This project is for recreating web designs pixel-accurately from reference screenshots as single `index.html` files using Tailwind CSS.

See `.claude/rules/` for the full ruleset:
- `iteration-workflow.md` — generate → screenshot → compare → fix loop
- `technical-defaults.md` — Tailwind CDN, placehold.co, single-file conventions
- `match-reference-exactly.md` — fidelity rules (don't add or "improve")
- `comparison-specificity.md` — how to describe visual mismatches
