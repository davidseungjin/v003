---
description: How to report visual mismatches when comparing screenshots to a reference design
---

When comparing screenshots, always be specific and quantitative about what's wrong. Examples:

- "Heading is 32px but reference shows ~24px"
- "Gap between cards is 16px but should be 24px"
- "Background color is #f0f0f0 but reference shows #f7fafc"
- "Border radius is 8px but reference shows 16px"

Never say "looks a bit off" — name the property, the current value, and the target value.
